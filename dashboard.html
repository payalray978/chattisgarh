Add a New Unit
Instruction to add new unit
  1. Udyam Aakanksha is a computer-generated acknowledgment and does not require a signature.
  2. It is issued as per Notification Number F20-4 / 2016 / (1) / 6 dated 06/02/2016.
  3. If a unit already has an Udyam Aakanksha, please avoid creating a duplicate.
  4. There is no approval process for Udyam Aakanksha. Users need to pay an application fee for registration based on the industry category:
    • Micro: INR 1,000
    • Small:INR 5,000
    • Medium: INR 10,000
    • Large: INR 15,000
  5. Fees paid for duplicate Udyam Aakanksha will neither be refunded nor transferred.
  6. If approvals are missing in an existing Udyam Aakanksha, please use the helpdesk option in the old Udyam Aakanksha to apply for the required approvals.
  7. For any assistance, contact the helpdesk support.
  8. Units with plant and machinery valued above INR 50 crore (Large Category) and total capital investment of INR 100 crore or more will automatically qualify for the "Invitation to Invest" program.
